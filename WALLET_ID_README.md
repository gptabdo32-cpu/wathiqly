# محفظة "وثّقلي" المتكاملة للهوية الرقمية والمدفوعات (Wathiqly ID & Pay Wallet)

تم تحديث منصة "وثّقلي" لتشمل نظاماً بيئياً متكاملاً يجمع بين المدفوعات الفورية والهوية الرقمية الموثقة، مع الالتزام بالمعايير العالمية في الهندسة والأمان.

## الميزات الجديدة

### 1. مدفوعات P2P الفورية
*   **تحويل مباشر**: إرسال واستقبال الأموال بين المستخدمين باستخدام أرقام الهواتف الموثقة فقط.
*   **سرعة وأمان**: تتم المعاملات بشكل فوري مع سجل تدقيق (Audit Log) كامل لكل حركة مالية.
*   **أمان مالي**: تتبع معايير PCI-DSS لضمان سلامة البيانات المالية.

### 2. دفع الفواتير والخدمات الرقمية
*   **تكامل الاتصالات**: دفع فواتير وشحن رصيد "ليبيانا" و"المدار" وخدمات LTT.
*   **الخدمات العامة**: واجهات جاهزة للتكامل مع فواتير الكهرباء والمياه والخدمات الحكومية.
*   **إيصالات موثقة**: إصدار إيصال رقمي لكل عملية دفع يمكن الرجوع إليه في أي وقت.

### 3. الهوية الرقمية الموثقة (SSO)
*   **بوابة دخول موحدة**: استخدام حساب "وثّقلي" كـ Single Sign-On للدخول إلى تطبيقات الطرف الثالث.
*   **بروتوكول OAuth 2.0**: تطبيق المعايير العالمية في مشاركة البيانات الموثقة بشكل آمن.
*   **مستويات التحقق**: تفعيل الميزات بناءً على مستوى التحقق (Level 1-3) لضمان أعلى درجات الثقة.

## الهندسة التقنية والأمان

### المعايير المتبعة
*   **ISO 27001**: تطبيق سياسات صارمة في تسجيل العمليات (Audit Logging) وتشفير البيانات.
*   **AES-256 Encryption**: تشفير البيانات الحساسة والملاحظات المالية.
*   **Atomic Transactions**: ضمان سلامة البيانات المالية عبر استخدام المعاملات الذرية في قاعدة البيانات لمنع حدوث أي خطأ في الأرصدة.

### كيفية الوصول
يمكنك الوصول للمحفظة الجديدة عبر المسار `/wallet-id` في التطبيق، حيث ستجد لوحة تحكم متكاملة تدير كافة احتياجاتك المالية والرقمية.
